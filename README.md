# 概览
云监控（CloudWatch）是一项专为UCloud云平台中产品及资源进行监控的服务，旨在帮助云上用户实时掌握云上资源的使用情况和运行状态，并及时对故障资源进行处理，保证业务正常运行。

云监控服务默认开通，借助云监控服务，您可以全面掌控UCloud云平台上的资源使用和业务运行状况。通过告警通知管理及灵活的监控模板配置，您能够及时发现云资源异常并迅速响应，保证您的服务及应用的良性运行。
