# __功能特性__

| 功能         | 说明                                                                                                                                                                                                                                                                                                    |
|--------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **产品监控** | 产品监控以云产品维度为核心，支持用户按产品类别（如云主机、云硬盘、弹性IP等）查询已购买资源的实时监控数据。系统默认集成关键指标（如CPU使用率、网卡出/入带宽、磁盘读/写次数等），无需手动配置即可快速查看资源状态。 |
| **Dashboard**    | Dashboard 提供灵活的可视化监控大盘，包含系统默认大盘与用户自定义大盘两种模式。<br/>● 默认大盘：系统预设常用云产品的典型监控图表组合，用户只需选择资源即可查看完整监控数据。<br/>● 自定义大盘：用户可根据业务需求自由组合图表、指标、资源和分组展示方式，打造个性化监控视图，实现多产品、多维度的统一监控。 |
| **监控看图**     | 监控看图提供灵活的实时查询功能，用户可在单一面板中自由选择云产品类型、项目、地域、指标及资源，并设置时间范围，动态生成监控图表。支持多面板组合，实现多维度数据对比分析。                                                                                      |
| **资源分组**     | 用户能够将同一云产品中同项目、跨地域的实例按业务需求进行分组管理。当用户创建大量资源时，可以按需求建立资源分组，便于后续更便捷的管理资源、设置告警策略，提升运维效率。                                                       |
| **告警管理**     | 告警管理模块，帮助用户实现从规则配置、触发控制到记录追踪的全流程告警管理。<br/>系统提供 **告警记录、告警策略、告警屏蔽、条件模板** 四个核心功能，支持灵活配置和高效维护，确保用户能够及时发现并处理云上异常，保障业务稳定运行。 |
| **进程监控** | 支持对云主机（UHost）和裸金属云主机（UPhost）上运行的关键业务进程进行持续性监控。（当前仅支持Linux系统） |
| **通知模板** | 告警通知模板用于统一配置告警通知的发送方式、接收对象及生效时间规则，并可在多个告警策略中复用，帮助您减少重复配置、提升告警管理效率。 |
