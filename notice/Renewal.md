# 产品更新动态

## 2025年12月

| 功能名称     | 功能描述                                                     |
| ------------ | ------------------------------------------------------------ |
| **进程监控** | 支持对云主机（UHost）和裸金属云主机（UPhost）上运行的关键业务进程进行持续性监控。<br />注：本次发布仅支持Linux系统。 |
| **通知模板** | 告警通知模板用于统一配置告警通知的发送方式、接收对象及生效时间规则，并可在多个告警策略中复用，帮助您减少重复配置、提升告警管理效率。 |
