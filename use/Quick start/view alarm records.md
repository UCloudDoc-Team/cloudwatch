# 查看告警记录
## 背景信息
告警历史可以展示自定义时间范围内所有告警规则的状态变化，您可以统一、方便地回溯和查看告警记录。

## 操作步骤
1. 登录云监控控制台。
2. 在顶部导航栏选择告警管理  > 告警记录，进入告警记录列表页。
3. 在告警记录页面，用户可以选择项目，并自定义时间范围，默认为最近1小时。
